ABOUT

Xuma is a decentralized database built with blockchain technologies. Xuma’s decentralized network connects data consumers and resource providers. Consumers can access a global decentralized database system with all the benefits of blockchain technology. Resource providers will be compensated for the pooled resources.

Masternode

Xuma Masternode divides up the many tasks in our decentralized networks. Masternode owners are incentivized to contribute resources to the network.

Smart Contracts

Allows consumers to define rules based on our market contract. Our smart contract will be able to work across blockchains and reduce if not eliminate data silos prevalent in today’s systems.

X-Swarm

A key technology to encrypt, distribute, store and retrieve data across Xuma’s global database network.

INSTRUCTIONS

Setting up swap space on a Ubuntu VPS
$ sudo fallocate -l 4G /swap
$ sudo chmod 600 /swap
$ sudo mkswap /swap
$ sudo swapon /swap
$ echo '/swap none swap sw 0 0' | sudo tee -a /etc/fstab
Compiling and running xumad on Ubuntu:
$ add-apt-repository ppa:bitcoin/bitcoin -y
$ apt-get update
$ apt-get upgrade -y
$ apt-get autoremove -y
$ apt-get install git build-essential libtool autotools-dev autoconf pkg-config libssl-dev libboost-all-dev libdb4.8-dev libdb4.8++-dev libminiupnpc-dev -y
$ mkdir ~/source && cd ~/source && git clone https://github.com/xumacoin/xuma-core.git
$ cd ~/source/xuma-core && ./autogen.sh && ./configure
$ make all install
$ xumad
$ xuma-cli getinfo
$ xuma-cli getbalance
$ xuma-cli getaccountaddress ""
Sample masternode xuma.conf file:
masternode=1
masternodeprivkey=<xuma-cli masternode genkey>
rpcuser=xumarpc
rpcpassword=create_a_random_password_here
staking=0
daemon=1
server=1
listen=1
txindex=1
Setting up a hot masternode:
Get the default wallet address that has a 0 balance: $ xuma-cli getaccountaddress ""
Or create a new wallet address that has a 0 balance: $ xuma-cli getaccountaddress "example"
Send 10,000 XMX to that address from your primary wallet: $ xuma-cli sendtoaddress XLd5zb7KJewkBNH7xb94r9WHUVrxKiXLcS 10000
Allow 15 confirmations and start the masternode: $ xuma-cli masternode start
